# Depot

framework for React apps. [changelog](https://github.com/pengfeiWang/depot-cli/blob/master/changelog.md)

# 安装

```
npm install depot-cli -g
```

# 使用


# 构建开发环境

```sh
depot cli myApp
```

`depot cli myApp`
`myApp` 为所要创建的目录名称
执行完毕后, 程序会创建  `myApp` 目录, 并生成开发所依赖的环境, 与文件结构

```sh
cd myApp
npm i
depot dev
```
